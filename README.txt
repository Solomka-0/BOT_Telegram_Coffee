Привет. Как и большинство подобных файлов, этот поможет вам разобраться в функционале и настройке бота. И так, для начала нужно вас познакомить. Это Coffee (можете поменять имя) - просто бот для нахождения собеседников. Предназначен для работы с телеграмом. Конечно, можете постараться запихнуть его в другую соц. сеть, но я могу пожелать вам только удачи. Так вот.. Пора разобраться со стандартными настройками. Глядите сюда, пообщавшись с BotFather вы можете ему задать:
- Имя (команда /setname): 'Coffee'/'Coffee Bot ☕️'.
- Описание (команда /setabouttext): 'Чат-бот. Помогает найти собеседников для вас'
- Что делает(команда /setdescription): 'Он может найти вам человека (живого)'
- Аватар (команда /setuserpic): можно найти в репозитории
- Список команд: 
subscibe - Подписаться
unsubscribe - Отписаться
info - Информация о подписчиках
find_couple - Ищет собеседника для вас
start - Выводит стартовое сообщение
help - Выводит вспомогательную информацию

Не забудьте создать бота и узнать у папочки токен!
Для нормальной работы его нужно вставить в data.py в подходящий раздел.
